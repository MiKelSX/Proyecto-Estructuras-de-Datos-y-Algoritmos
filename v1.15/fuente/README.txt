Proyecto Estructuras de Datos y Algoritmos

Este proyecto, titulado "Optimización de Búsqueda en Texto", es parte de un trabajo académico universitario, enfocado en el estudio y análisis de estructuras de datos y algoritmos para la búsqueda eficiente de patrones en textos.

Contenido del Proyecto:
- `fuente`: Carpeta que contiene los archivos fuente del proyecto.
- `informe`: Carpeta que contiene el informe del proyecto en formato PDF.
- `proyecto.tex`: Archivo LaTeX del informe.
- `hawking-stephen-historia-del-tiempo.txt`: Archivo de texto de ejemplo para pruebas.
- `biblia.txt`: Archivo de texto de ejemplo para pruebas.

Ejecución del Proyecto:
- Consultar el archivo `EXECUTE` para obtener instrucciones detalladas sobre cómo ejecutar el proyecto y utilizar el cliente interactivo.
- Asegúrarse de tener Python 3.13 instalado en el sistema antes de ejecutar el proyecto.
