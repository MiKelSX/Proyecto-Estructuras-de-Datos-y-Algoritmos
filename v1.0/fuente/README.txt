# Búsqueda de Patrones en Texto

Este programa permite buscar patrones en un archivo de texto utilizando dos métodos:
1. Búsqueda por Fuerza Bruta
2. Búsqueda Indexada con Tabla de Hash

## Requisitos

- Python 3

## Instrucciones

1. Asegúrate de tener el archivo `proyecto.tex` en el mismo directorio que los scripts.
2. Ejecuta el programa principal con el siguiente comando:

```bash
python fuente/src/cliente_interactivo.py
